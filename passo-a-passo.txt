CONTROLE DE VERSÃO 
- Mantém base histórica da evolução do codigo donte de um software
- Ferramentas:
 --Git

Passos:

0 - Instalar o Git
1 - Escolher uma pasta que será o repositório
2 - "git init" para iniciar um novo repositório do git (dir, mkdir e cd)
3 - "git status" para verificar o estado dos arquivos no repositorio
4 - "git add" para incluir arquivos no tracking do git
5 - "git config -- global user.name "seu nome" 
6 - "git config -- global user.email "seu@email.com" 
7 - "git commit -m" "uma mensagem descritiva para confirma as mudanças do stage para o repositorio local
8 - "git log" para ver o historico de comits do repositorio local
9 - "git clone URL" para baixar um repositorio remoto rio local
